## Про проєкт

**Cosmic Explorer** — це одностронницовий сайт (SPA-like), який у доступній формі розповідає про:

- Дорогу Молочну та її масштаби
- 8 планет Сонячної системи
- Темну матерію та темну енергію
- Чорні діри та їх будову

Сайт містить анімації появи елементів при прокрутці, адаптивну навігацію та систему реєстрації/входу користувача.

---

## Функціонал

- **Навігація** — зручне меню з прокруткою до секцій сторінки
- **Секція планет** — картки всіх 8 планет Сонячної системи з описами
- **Освітній контент** — секції про Дорогу Молочну, таємниці космосу та чорні діри
- **Анімації** — плавна поява елементів при скролі (ScrollReveal)
- **Реєстрація / Вхід** — форми автентифікації з валідацією (дані зберігаються в `localStorage`)

## Технології

| Технологія | Призначення |
|---|---|
| HTML5 | Розмітка сторінок |
| CSS3 | Стилізація та адаптивний дизайн |
| JavaScript (Vanilla) | Інтерактивність, автентифікація |
| [ScrollReveal](https://scrollrevealjs.org/) | Анімації появи елементів при прокрутці |
| localStorage | Зберігання даних користувача на стороні клієнта |

## Запуск

Проєкт не потребує збірки або сервера — достатньо відкрити `index.html` у браузері.

```bash
# Клонувати або розпакувати архів проєкту
# Відкрити у браузері:
open index.html
```

> **Примітка:** для коректної роботи шляхів до зображень рекомендується відкривати через локальний сервер (наприклад, розширення Live Server у VS Code).
